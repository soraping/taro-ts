### TARO ts 版

> 开始

```bash
# 下载全局库
npm install -g @tarojs/cli
# or
yarn global add @tarojs/cli

# 安装依赖
yarn

# 启动项目
yarn start
```

> 目录结构

```bash
├── actions     # redux action文件目录
├── app.less    # 主样式
├── app.tsx     # 入口
├── constants   # 常量
├── index.html
├── interface   # 类型文件
├── pages       # 模块目录
├── reducers    # redux reducer文件目录
├── static      # 小程序静态文件目录
├── store       # redux store
├── styles      # 样式文件目录
└── utils       # 工具目录
```
